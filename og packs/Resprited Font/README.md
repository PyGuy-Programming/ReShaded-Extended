# Resprited

A Minecraft resource pack
created by **Infernalizard**

---

### Description
Resprited aims to give a new look to Minecraft while still keeping its core feeling we all know and love.

---

## Installation
1. Download the latest version from the [Modrinth page](https://modrinth.com/resourcepack/resprited_infernalizard).
2. Place the `.zip` file into your `resourcepacks/` folder.
3. Launch Minecraft and enable it in the settings.
